# macOS アプリにリフレッシュボタンを追加

## 概要・やりたいこと
macOS メニューバーアプリで最新のデータを取得するために、毎回ウィンドウを閉じて再度開く必要がある。「Today」表示時にリフレッシュボタンを追加し、ワンクリックでデータを再取得できるようにする。

## 前提・わかっていること
- データ再取得は `viewModel.loadTodos()` を呼ぶだけで完了する（仕組みはすでにある）
- 自動リロードはパネル表示時・60秒タイマー・日付切替時に走っているが、手動トリガーがない
- ヘッダーの構造: `ZStack` で左に `<`、中央に日付テキスト、右に `>` を配置。今日のときは `>` が非表示
- `KeyablePanel` で `canBecomeKey = true` にしているため、SwiftUI の `keyboardShortcut` が使えるはず
- ナビゲーションボタンは 20pt。リフレッシュボタンはそれより少し大きめ（24pt）にする

### 設計決定事項
| 項目 | 決定 |
|---|---|
| 配置 | 「Today」テキストの右、今日のみ表示 |
| アイコン | `arrow.clockwise`（24pt） |
| フィードバック | 回転アニメーション |
| ショートカット | `Cmd+R`（SwiftUI ネイティブ） |
| 非今日 | ボタン非表示 |
